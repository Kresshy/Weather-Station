# Weather Station v3.6.9 - The Hardware Readiness Update

## ✨ New Features & Improvements
*   **Proactive Bluetooth Activation**: The app now automatically requests to enable the Bluetooth adapter at startup and during resume cycles if it is currently disabled. 
*   **Location Services Monitoring**: Added a system-level check for Location Services (GPS/Network). Since Bluetooth discovery requires active Location Services on most Android versions, the app now detects if they are disabled and provides a direct shortcut to the system settings via a Snackbar notification.

## 🛠️ Bug Fixes & Refinements
*   **Unified Hardware Check**: Consolidated Bluetooth and Location status monitoring into a single `checkHardwareStatus()` workflow in `WSActivity`, ensuring consistency between permissions and hardware state.

## 🧪 Quality & Tests
*   **Full Suite Verification**: All 110+ unit tests passed, confirming the integrity of the connectivity plane and data analysis layers.
*   **Build Integrity**: Verified through the mandatory sequence: `./gradlew spotlessApply test build`.
