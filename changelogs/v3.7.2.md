# Weather Station v3.7.2 - The Task Removal Persistence Update

## 🛠️ Bug Fixes & Improvements
*   **Reliable Bluetooth Auto-Disable**: Resolved a limitation where Bluetooth would not be disabled if the user swiped the app away from the "Recent Apps" view. 
*   **Task Removal Hook**: Implemented the `onTaskRemoved` lifecycle hook in `WeatherService`. The background service now detects when the application process is being terminated via the task manager and ensures that the "Disable Bluetooth on Quit" preference is honored, even when the "Quit" button is not used.
*   **Enhanced Service Resilience**: Integrated `SharedPreferences` into the `WeatherService` to provide real-time access to user hardware preferences during the final process termination sequence.
