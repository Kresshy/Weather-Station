# Weather Station v3.6.8 - The Unified Handover Update

## 🛠️ Bug Fixes & Refinements
*   **Resolved Cold Start Deadlock**: Fixed a critical issue where an initial connection to a BLE device (misidentified as "Unknown" by Android) would trap the app in a failing Classic Bluetooth handshake, ignoring subsequent BLE discovery updates.
*   **Technology Upgrade Logic**: Implemented a "Driver Upgrade" mechanism in `CompositeConnection`. The system now correctly prioritizes specific technology info (BLE) over generic guesses (Classic), even during an active connection attempt.
*   **Multi-Station Switching**: Hardened the handover logic to ensure that switching between different weather stations (even with the same driver type) correctly resets the underlying driver and clears hardware resources before initiating the new link.
*   **Atomic State Management**: Unified the handover, upgrade, and redundancy checks into a single decision matrix, ensuring mathematical certainty in driver selection.
