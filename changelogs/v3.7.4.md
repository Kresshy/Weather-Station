# Weather Station v3.7.4 - The Multi-Generation Compatibility Update

*   **Legacy Android Compatibility**: Resolved the "not drawing" issue on Android 12 and older (API < 33). Implemented the legacy `onCharacteristicChanged` callback to ensure data capture on older Bluetooth stacks.

*   **Discovery Resilience**: Hardened the discovery plane to prevent "type downgrading" when a device is found via multiple scanning paths simultaneously.
