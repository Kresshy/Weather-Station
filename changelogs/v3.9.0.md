# Weather Station v3.9.0 - The Visual Flow Update

## 🚀 New Features & Enhancements
*   **Right-to-Left Chart Filling**: Refactored the visualization engine to fill data from the right side of the chart. New data points now appear at the right edge and move left, providing a more intuitive "live stream" feel.
*   **Constant Scale Visualization**: Fixed X-axis stretching; the charts now maintain their full time-window scale (e.g., 5 minutes) from the moment the first data point arrives.
*   **Consolidated v3.8.0 Features**:
    *   **Sliding Window Weather Data (US001)**: Time-based historical data pruning.
    *   **Restore File Logging (US002)**: Asynchronous internal storage logging.
    *   **Wind Speed Calibration (US003)**: Linear formula for custom carbon hardware.
    *   **Initial Point Alignment & Precision (US004)**: 2-decimal UI and bottom-corner anchoring.

## 🛠️ Bug Fixes & Refinements
*   **Resolved X-Axis Stretching**: Prevented the initial few data points from being pulled to the full width of the chart.
*   **Session Anchor Hardening**: Fixed a potential crash when initializing session starting points.
*   **Build Integrity**: Verified through the mandatory sequence: `./gradlew spotlessApply test build`.

## 📦 Deliverables
*   **WeatherStation_v3.9.0_debug.apk**: Signed debug APK with the complete visual and precision feature set.
*   **weatherstation.ino v2.1**: High-precision Arduino firmware.
