# Changelog v3.5.0 (Connectivity & UX Refactor)

## 📡 Bluetooth Connectivity (Pairing & Security)
*   **In-App Pairing Support**: Implemented `createBond()` support, allowing users to pair with new weather stations directly from the device list without leaving the application.
*   **PIN Entry Support**: Added a Material PIN entry dialog that automatically triggers when a station requests a security code.
*   **Pairing State Management**: Integrated `ACTION_BOND_STATE_CHANGED` and `ACTION_PAIRING_REQUEST` receivers to provide real-time feedback during the bonding process.

## 📱 User Interface (Pro-Pilot Experience)
*   **Sectioned Device List**: Replaced the legacy `ListView` with a high-performance `RecyclerView`.
*   **Smart Grouping**: The Bluetooth list now automatically separates "Paired Devices" (top) from "Available Devices" (bottom) using visual headers.
*   **Seamless Transition**: Devices automatically move to the "Paired" section upon successful bonding.
*   **UI Modernization**: Updated `BluetoothDeviceRecyclerAdapter` for smoother scrolling and better view recycling on legacy hardware.

## ⚙️ Logic & Architecture
*   **Calibration Centralization**: Removed the redundant manual calibration feature from the Android app. Calibration is now handled as a "Source of Truth" in the Arduino firmware, ensuring absolute wind speed safety thresholds (e.g., the 5.0 m/s limit) are consistently applied.
*   **Domain Expansion**: Introduced `PairDeviceUseCase` to encapsulate pairing logic, further refining the Clean Architecture implementation.
*   **Codebase Cleanup**: Purged all calibration-related strings, layouts, and data constants. Removed `.DS_Store` artifacts from the repository.

## 🧪 Quality Control & Validation
*   **Bluetooth Manager Tests**: Added `WeatherBluetoothManagerTest` to verify pairing and PIN delegation.
*   **Regression Prevention**: Updated `WeatherViewModelTest` and `WeatherRepositoryImplTest` to reflect the removed calibration logic.
*   **Mandatory Sequence**: All changes verified via `./gradlew spotlessApply test build`.
